Some Things I've Worked On / Am Working On Independently

These are just a couple of my projects — my GitHub is most likely up to date with my latest activity.

  • This website! The original version used PHP, but I have since rewritten the codebase to use Angular!
  • From Scratch Database: A recreation of a SQL database using Rust to create, read, update, and delete from CSV files. I'm aiming to have this database do almost anything a SQL database can do, including but not limited to full joins, aggregations, SQL math, aliasing, and transactions. Possibly even more functions that I may or may not need in some upcoming projects.
  • ChessWeb: A Django web app for Oakwood Elementary's Chess Club that uses a SQLite database to hold player information such as their name and rating (ELO). This program uses a greedy algorithm to pair players based on their ratings and updates their ratings based on the results of their previous games. You can find the code on my GitHub — it's in pre-release!
  • 3D Visualizer: A 3D plotter for prey-predator equations, using Three.js to approximate the populations with either Euler's Method or Runge-Kutta 4. This was completed as my final project with Brynlee Thain for our Differential Equations class! You can either visit GitHub for the source code or go to the web version.
