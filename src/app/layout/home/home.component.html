Some Things I've Worked On / Am Working On Independently

These are just a couple of my projects — my GitHub is most likely up to date with my latest activity.

  • I first started my journey with computer science by building a simple website, which later turned into version 1.0.0 of this website. It was completely built in PHP without any file structure (no CodeIgniter or Laravel), just vanilla PHP. Since working at Ken Garff, where all development is done with Angular and Nest.js, I rebuilt this website to use Angular. Currently, this is hosted on a DigitalOcean droplet with Apache/Ubuntu.
  • For my 2025 summer project, I'm working on a Rust Improved of SQL (RISQL), which aims to do everything a SQL database can do while also allowing me to implement custom features like encryption and SQL injection protection. As this project is in early development, you can follow my progress on GitHub. I'm hopeful that I can get RISQL finished by the end of summer, giving me time to make improvements and fix any bugs I find.
  • Due to my work as Co-Director at my local elementary school's chess club, I wanted to create chess ratings and pairings software which would save the director or volunteer doing the ratings and pairings up to 4 hours weekly. Thus, I made ChessWeb. There are two versions of ChessWeb: the one built with LAMP as my final project for my databases class (a simple CRUD application), but it lacked the pairing algorithm. As an improvement on ChessWeb 1.0, I built ChessWeb 2.0, which uses the Django framework. Currently, this code is in pre-release, and I'm planning to get it fully deployed for my local club by the end of May 2025.
  • I created this 3D Visualizer for my differential equations class after finding the original tool to be rather slow and clunky. Although this tool cannot yet evaluate fractions, sines, or cosines, it works well for giving either Euler or Runge-Kutta 4 approximations for predator-prey systems of equations. You can visit the web version here.
