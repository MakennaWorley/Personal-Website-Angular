Some Things I've Worked/Working On Independently

These are just a couple of my projects I'm working on, my Github is most likely up to date with my latest activity.

  • This website! The original version of this website used PHP but I have since rewritten the codebase to use Angular!
  • From Scratch Database: A recreation of a SQL database using Rust to create, read, update, and delete from csv files. I'm aiming to have this database do almost anything a SQL database can do including but not limited to full joins, aggregations, SQL math, aliasing, and transactions. Possibly even more functions that I may or many not need in some upcoming projects.
  • ChessWeb: A Django webapp for the Oakwood Elementary's Chess Club which uses a SQLite database to hold player information such as their name and rating (ELO). This program uses a greedy algorithm to pair players based on their rating and updates their ratings based on the result of their previous games. You can find the code on my GitHub, it's in pre-release!
  • 3D Visualizer: A 3d plotter for prey and predator equations making use of three.js to aproximate the populations using either Euler's Method or Runge Kutta 4. This was completed as my final project with Brynlee Thain for our Differential Equations class! You can either visit GitHub for the source code or go to the web version
