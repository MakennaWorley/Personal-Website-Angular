From Scratch Database: A Rust recreation of a SQL Database

This is a side project I spun up on a weekend after deciding it would be fun to recreate a database in Rust to not only improve my own Rust coding ability but also learn how SQL databases work! The final goal is to have this project able to do anything a regular SQL database can do, but this uses .csv files at the moment (I might look into how SQL stores data to recreate that too). But I might also add in some custom functions for any future projects or have some SQL injection protection built in. Who knows!

ChessWeb: A Chess Rating and Pairing System

This is a custom made Django web application that was designed from the ground up to save around 4+ hours a week by automating the process of updating ratings and pairing the student in Oakwood Elementary's Chess Club students. This program usings a greedy algorithm for pairing the players based on their rating and previous opponents.

Currently, the first prerelease is out on my GitHub! It requires the user to know how to start up a Django python project and has limited functionality. I plan on finishing up the full functionality soon! When it is published for release, I'll also make an announcement on my LinkedIn and have an installation guide for getting the project set up on your own machine.

3D Visualizer: Differential Equations Visualizer

Built using Three.js, this code uses Euler and Runge Kutta 4 to approximate the system of equations (prey and predator by default). This shows both the table for the values and either 3D or 2D graphs depending on the angle the user wants to view the graph. Built at a final project for Differential Equations with Brynlee Thain. This code contains either the js version meant for running locally or on the server and a python script to view locally.
